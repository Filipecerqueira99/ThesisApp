# backend - server

## Project setup
```
npm install

node server.js
```


## Note
```
Make sure you have the db.config.json with your username and password from MySQL

Make sure, on your MySQL to have a table for users (uname, password)

Add required database information to db.config.json

    "connectionLimit": ...,
    "database": ...,
    "debug": ...,
    "host": ...,
    "password": ...,
    "user": ...,
    "port": ...
    
```
